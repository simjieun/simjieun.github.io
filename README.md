# Joy's Dev Blog

개발자 Joy의 기술 블로그입니다. 학습한 내용과 경험을 기록하고 공유합니다.

🔗 **[https://simjieun.github.io](https://simjieun.github.io)**

## 소개

- 공부하고 경험한 내용을 꾸준히 기록하는 개발 블로그
- 프론트엔드 개발, 트러블슈팅, 성장 과정을 주로 다룹니다

## 기술 스택

- **Framework**: Jekyll + [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) 테마
- **Hosting**: GitHub Pages

## 로컬 실행

```bash
bundle install
bundle exec jekyll serve
```
